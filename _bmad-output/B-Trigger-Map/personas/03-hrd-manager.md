## Profile Summary

**Budi, 38 tahun, HRD Manager di PT Retail Fashion (150+ karyawan, 5 cabang)**

Budi adalah HRD Manager yang handle payroll, leave management, attendance monitoring, dan employee data. Setiap bulan dia spend 2 hari untuk manual pull attendance data dari ZKTeco machines di 5 cabang, calculate lembur/cuti di Excel, dan generate payslips. Dia tahu ada cara lebih efficient, tapi stuck dengan manual process dan disconnected tools.

**Current Reality:** 2 hari payroll processing, manual attendance compilation, Excel-based calculation  
**Desired Reality:** 4 jam payroll processing, auto-sync attendance, automated calculation  
**Transformation:** Manual payroll processor → Strategic HR partner

---

## Background

Budi graduated dari Psikologi 15 tahun lalu, join PT Retail Fashion 8 tahun lalu sebagai HR Staff. Naik jadi HRD Manager 3 tahun lalu ketika company expand ke 5 cabang. Dia responsible untuk:

- Monthly payroll (150+ employees)
- Leave management (cuti, sakit, lembur)
- Attendance monitoring (5 ZKTeco machines)
- Employee data management
- Recruitment & onboarding

Company pakai ZKTeco untuk attendance, tapi data harus manual pull dari tiap machine, compile di Excel, calculate lembur/cuti manual, dan generate payslips satu-satu. Budi spend 2 hari setiap bulan untuk payroll processing, bukan strategic HR work.

---

## Current Situation

**Pain Points:**
1. **Manual Attendance Compilation:** Pull data dari 5 ZKTeco machines → compile di Excel → reconcile discrepancies
2. **Manual Payroll Calculation:** Calculate lembur/cuti manual → apply tax/BPJS → generate payslips satu-satu
3. **Slow Leave Approval:** Leave requests via WhatsApp → manual check balance → manual approve → manual update Excel
4. **No Real-Time Visibility:** Can't see real-time attendance across cabang, harus wait for month-end compilation
5. **Error-Prone:** Manual calculation = high error risk = employee complaints

**Current Workflow:**
- Day 1-2 (Month-end): Pull attendance data dari 5 machines → compile di Excel → reconcile
- Day 3-4: Calculate lembur/cuti → apply tax/BPJS → generate payslips
- Day 5: Distribute payslips → handle employee questions → fix errors
- Throughout month: Manual leave approval via WhatsApp → update Excel

**Frustration:** "Saya jadi data entry clerk, bukan HR strategist. Padahal saya bisa fokus ke talent development kalau punya waktu."

## 6 Driving Forces

### ✅ WANTS (Positive Drivers)

**1. Want: Automated Payroll Processing**
- **AkuBook Promise:** Payroll processing < 4 jam (vs 2 hari) — attendance auto-sync, lembur/cuti auto-calculate, payslips auto-generate
- **How:** ZKTeco integration → real-time attendance sync → auto-calculate lembur/cuti → one-click payroll generation
- **Impact:** 2 hari freed up untuk strategic HR work (talent development, culture building)

**2. Want: Real-Time Attendance Visibility**
- **AkuBook Promise:** See real-time attendance across all cabang — who's in, who's late, who's on leave
- **How:** Dashboard dengan real-time attendance status, alerts untuk late/absent, drill-down per cabang/department
- **Impact:** Proactive attendance management, no more month-end surprises

**3. Want: Fast Leave Approval**
- **AkuBook Promise:** Leave requests via mobile app → auto-check balance → one-click approve → auto-update attendance
- **How:** Mobile app untuk employee submit leave → manager approve via notification → system auto-update balance
- **Impact:** Employee satisfaction, no more WhatsApp chaos, transparent leave balance

### ❌ FEARS (Negative Drivers)

**4. Fear: ZKTeco Integration Complexity**
- **AkuBook Answer:** Seamless ZKTeco integration — plug-and-play, no custom development, existing machines work as-is
- **How:** Built-in ZKTeco connector, step-by-step setup wizard, test mode untuk verify before go-live
- **Impact:** No hardware replacement cost, no downtime, existing investment preserved

**5. Fear: Payroll Accuracy**
- **AkuBook Answer:** Automated calculation eliminates human error — tax/BPJS rules built-in, validation checks, audit trail
- **How:** Pre-configured tax/BPJS rules, validation alerts untuk anomalies, detailed calculation breakdown per employee
- **Impact:** Zero payroll errors, employee trust, compliance confidence

**6. Fear: Employee Resistance**
- **AkuBook Answer:** Employee-friendly mobile app — easy clock in/out, transparent payslip, self-service leave request
- **How:** Mobile app dengan face recognition, digital payslip dengan calculation breakdown, leave balance visibility
- **Impact:** Employee adoption, no resistance, positive feedback
