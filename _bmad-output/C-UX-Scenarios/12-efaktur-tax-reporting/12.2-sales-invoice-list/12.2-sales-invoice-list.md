# Page 12.2: Sales Invoice List (Unprocessed)

**Scenario:** 12 - e-Faktur & Tax Reporting  
**Page:** 12.2 - Sales Invoice List (Unprocessed)  
**Created:** 2026-05-13  
**Status:** ✅ Specified

---

## Page Purpose

Select sales invoices for batch e-Faktur generation

---

## Key Components

### 1. Filter Bar
- Status: Unprocessed / Processed / All
- Date Range: Invoice date
- Customer: Filter by customer
- NPWP Status: Valid / Invalid / Non-PKP

### 2. Invoice Table
Columns: ☑ | Invoice# | Date | Customer | NPWP | Amount | PPN | Status

### 3. Batch Actions
- Select All (checkbox)
- Generate e-Faktur Batch (primary button)
- Export List

### 4. Invoice Row
- Checkbox for selection
- Invoice number (linked)
- Invoice date
- Customer name
- NPWP (validated - green checkmark or red X)
- Invoice amount (DPP)
- PPN amount (11% or 12% based on date)
- Status badge (Unprocessed / Processed)

### 5. Validation Indicators
- Green checkmark: NPWP valid, ready for e-Faktur
- Red X: NPWP invalid, cannot generate e-Faktur
- Gray icon: Non-PKP customer, e-Faktur not required

---

## Interactions

- Select invoices → Enable "Generate Batch" button
- "Generate Batch" → Navigate to e-Faktur Generation (12.3)
- Click invoice → Open invoice detail modal
- NPWP validation → Real-time check on load
