## Components

### 1. Header
- **Logo**: AkuBook logo (top left)
- **Progress**: "Step 1 of 7" (top right)

### 2. Welcome Section
- **Heading**: "Selamat Datang di AkuBook! 👋"
- **Subheading**: "Mari kita setup perusahaan Anda dalam beberapa langkah mudah"

### 3. Form Fields

**Company Name (required)**
- Type: Text input
- Placeholder: "PT Nama Perusahaan Anda"
- Validation: Min 3 characters, required

**Industry Type (required)**
- Type: Dropdown select
- Options:
  - Distributor (sound, lighting, LCD)
  - Retail (toko, minimarket)
  - Manufacturing (pabrik, produksi)
  - Service (bengkel, jasa)
  - Lainnya
- Default: None (must select)

**Number of Employees**
- Type: Dropdown select
- Options: 1-10, 11-50, 51-100, 100+
- Default: 1-10

**Number of Warehouses/Branches**
- Type: Dropdown select
- Options: 1 lokasi, 2-3 lokasi, 4-5 lokasi, 6+ lokasi
- Default: 1 lokasi

### 4. Module Preview (Info Box)
- **Icon**: 💡 (lightbulb)
- **Text**: "Berdasarkan pilihan Anda, kami akan mengaktifkan modul:"
- **Module List** (dynamic based on selections):
  - ✓ Accounting (always)
  - ✓ Multi-Warehouse Inventory (if warehouses > 1)
  - ✓ Sales & Purchasing (always)
  - ✓ Attendance & HRM (if employees > 10)

### 5. Primary Action
- **Button**: "Lanjutkan →"
- **State**: Disabled until required fields filled
- **Action**: Validate inputs → Save to session → Navigate to Page 2

---

## Interactions

### On Load
1. Display welcome message
2. Focus on Company Name field
3. Module preview shows default modules (Accounting, Sales & Purchasing)

### On Industry Selection
1. Update module preview based on industry
2. If "Distributor" → highlight Multi-Warehouse
3. If "Manufacturing" → add Manufacturing module to preview

### On Employee Count Change
1. If 100+ → add HRM module to preview
2. If < 100 → remove HRM from preview (optional module)

### On Warehouse Count Change
1. If > 1 → add Multi-Warehouse to preview
2. If = 1 → show "Single Warehouse" in preview

### On Submit (Lanjutkan)
1. Validate required fields
2. Show loading state on button
3. Save data to setup session
4. Navigate to Page 2 (Company Profile)

---

## Validation Rules

**Company Name:**
- Required
- Min 3 characters
- Max 100 characters
- Error: "Nama perusahaan harus diisi (min 3 karakter)"

**Industry Type:**
- Required
- Must select from dropdown
- Error: "Pilih jenis industri Anda"

**Employee Count:**
- Optional (has default)
- No validation needed

**Warehouse Count:**
- Optional (has default)
- No validation needed

## Edge Cases

**Empty state:**
- All fields empty → button disabled
- Show placeholder text in inputs

**Validation errors:**
- Show inline below field
- Keep button enabled (allow retry)
- Focus on first error field

**Session timeout:**
- Auto-save form data every 30 seconds
- On return, restore previous values

**Browser back:**
- Warn user: "Setup belum selesai. Yakin ingin keluar?"
