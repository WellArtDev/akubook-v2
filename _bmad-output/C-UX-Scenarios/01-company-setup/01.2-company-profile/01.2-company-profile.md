## Components

### 1. Header
- **Logo**: AkuBook logo (top left)
- **Progress**: "Step 2 of 7" (top right)

### 2. Page Title Section
- **Heading**: "Profil Perusahaan"
- **Subheading**: "Informasi ini akan muncul di dokumen resmi (faktur, laporan, surat jalan)"

### 3. Form Fields

**Company Legal Name (required, pre-filled)**
- Type: Text input
- Value: Carried from Page 1
- Editable: Yes
- Hint: "(sudah terisi dari halaman sebelumnya)"
- Validation: Min 3 characters

**NPWP (required)**
- Type: Text input with mask
- Format: 00.000.000.0-000.000
- Auto-format: Add dots/dashes as user types
- Hint: "Format: 00.000.000.0-000.000"
- Validation: Must match NPWP format (15 digits)

**Full Address (required)**
- Type: Textarea (3 rows)
- Placeholder: "Jl. Nama Jalan No. 123, RT/RW, Kelurahan, Kecamatan"
- Validation: Min 10 characters

**City/Regency (required)**
- Type: Text input
- Placeholder: "Jakarta"
- Validation: Min 3 characters

**Postal Code (optional)**
- Type: Text input (numeric)
- Placeholder: "12345"
- Validation: 5 digits (if filled)

**Phone (required)**
- Type: Text input
- Placeholder: "021-1234567"
- Validation: Min 8 characters

**Email (required)**
- Type: Email input
- Placeholder: "admin@company.id"
- Validation: Valid email format

**Company Logo (optional)**
- Type: File upload
- Accepted: PNG, JPG, JPEG
- Max size: 2MB
- Preview: Show thumbnail after upload
- Actions: Upload or Skip

### 4. Navigation Actions
- **Back Button**: "← Kembali" (ghost style)
- **Primary Button**: "Lanjutkan →"
- **State**: Primary disabled until required fields valid

---

## Interactions

### On Load
1. Pre-fill company name from Page 1
2. Focus on NPWP field
3. Show format hint below NPWP

### NPWP Input Formatting
1. User types digits only
2. System auto-adds dots and dashes
3. Example: "123456789012345" → "12.345.678.9-012.345"
4. Validate format on blur

### Logo Upload
1. Click "Upload Logo" → file picker
2. Drag & drop supported
3. On select → validate size/format
4. Show preview thumbnail
5. "Lewati" button → skip logo, enable Continue

### On Back Button
1. Save current form data to session
2. Navigate to Page 1
3. Restore Page 1 with saved data

### On Submit (Lanjutkan)
1. Validate all required fields
2. Show loading state
3. Save to setup session
4. Navigate to Page 3 (Fiscal Settings)

## Validation Rules

**Company Legal Name:**
- Required
- Min 3 characters
- Error: "Nama perusahaan harus diisi"

**NPWP:**
- Required
- Must be 15 digits in format 00.000.000.0-000.000
- Error: "NPWP tidak valid (harus 15 digit)"

**Address:**
- Required
- Min 10 characters
- Error: "Alamat lengkap harus diisi"

**City:**
- Required
- Min 3 characters
- Error: "Kota/Kabupaten harus diisi"

**Postal Code:**
- Optional
- If filled: must be 5 digits
- Error: "Kode pos harus 5 digit"

**Phone:**
- Required
- Min 8 characters
- Error: "Nomor telepon tidak valid"

**Email:**
- Required
- Valid email format
- Error: "Email tidak valid"

**Logo:**
- Optional
- If uploaded: PNG/JPG, max 2MB
- Error: "File terlalu besar (max 2MB)" or "Format tidak didukung (gunakan PNG/JPG)"

## Edge Cases

**Invalid NPWP format:**
- Show error on blur
- Highlight field in red
- Keep Continue button disabled

**Logo upload fails:**
- Show error message
- Allow retry or skip
- Don't block Continue button

**Session timeout:**
- Auto-save every 30 seconds
- Restore on return

**Browser back:**
- Same as Back button (save and navigate)

**Pre-filled name edited:**
- Allow edit
- Validate on blur
- Update session
