## Page Purpose

Select bank account to reconcile from filtered list with detailed status

---

## Key Components

### 1. Filter Bar
- Status: All / Pending / Reconciled / Overdue
- Bank: All / BCA / Mandiri / BNI / etc.
- Date Range: Last reconciliation date
- Search: Account number or name

### 2. Account Table
Columns: Bank | Account | Balance | Last Reconciled | Unreconciled Items | Status | Actions

### 3. Bulk Actions
- Reconcile Multiple (checkbox selection)
- Export List
- Print Report

### 4. Account Row
- Bank icon + name
- Account number (masked: ****7890)
- Current balance (bold, large)
- Last reconciled date
- Unreconciled count (badge)
- Status indicator (green/yellow/red)
- Actions: Import | View | Reconcile

---

## Interactions

- Click row → Navigate to Import Statement (11.3)
- Click "Reconcile" → Navigate to Import Statement (11.3)
- Click "View" → Open transaction list modal
- Filter change → Update table immediately
- Checkbox select → Enable bulk actions

---

## Data API

```
GET /api/cash-bank/accounts?
  status={status}&
  bank={bank}&
  date_from={date}&
  date_to={date}&
  search={query}

Response: Array of accounts with reconciliation status
```

---

## Visual Style

- Table layout with alternating row colors
- Status badges (colored pills)
- Hover effect on rows
- Sticky header on scroll
- Pagination: 10 accounts per page
