## Reconciliation Lock

Once marked as reconciled:
- Status changes to "Reconciled"
- Transactions cannot be unmatched
- Adjustments cannot be modified
- Audit trail preserved
- Can be reopened only by Finance Manager

---

## Interactions

- Balance mismatch → Show error, prevent reconciliation
- [Mark as Reconciled] → Confirm dialog → Lock → Navigate to dashboard
- [Print Report] → Generate PDF → Open print dialog
- [Export] → Generate Excel → Download file
