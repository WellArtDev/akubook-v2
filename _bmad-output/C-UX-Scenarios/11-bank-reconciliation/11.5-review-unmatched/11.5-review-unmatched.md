# Page 11.5: Review Unmatched

**Scenario:** 11 - Bank Reconciliation  
**Page:** 11.5 - Review Unmatched  
**Created:** 2026-05-13  
**Status:** ✅ Specified

---

## Page Purpose

Resolve unmatched transactions through manual matching or creating entries

---

## Key Components

### 1. Unmatched Summary
- Total unmatched: 4 transactions
- Bank charges: 2 (auto-create suggested)
- Missing receipts: 1 (manual match needed)
- Timing differences: 1 (mark in transit)

### 2. Unmatched Transactions List

Each transaction shows:
- Bank transaction details
- Suggested action (badge)
- Action buttons

**Transaction Types:**

**Bank Charges:**
- Suggested: Auto-create journal entry
- Action: [Create Entry] button
- Preview: DR: Bank Charges, CR: Cash

**Interest Income:**
- Suggested: Auto-create journal entry
- Action: [Create Entry] button
- Preview: DR: Cash, CR: Interest Income

**Missing Receipts:**
- Suggested: Manual match
- Action: [Search & Match] button
- Opens search modal with system transactions

**Timing Differences:**
- Suggested: Mark as in transit
- Action: [Mark In Transit] button
- Will auto-match next period

### 3. Manual Match Interface

**Search Panel:**
- Search by: Amount, Date Range, Reference, Description
- Results table: System transactions not yet matched
- Drag-and-drop or click to match

**Match Preview:**
- Shows bank transaction + selected system transaction
- Confidence indicator
- Confirm match button

### 4. Bulk Actions
- Create All Bank Charges
- Mark All In Transit
- Export Unmatched List

---

## Interactions

- [Create Entry] → Auto-create journal entry → Remove from unmatched
- [Search & Match] → Open search modal → Select transaction → Confirm match
- [Mark In Transit] → Flag as in transit → Remove from unmatched
- All resolved → Navigate to Reconciliation Report (11.6)
