---

## Matching Logic

**Tier 1 - Exact Match (100%):**
- Amount + Date + Reference Number exact match

**Tier 2 - High Confidence (90-99%):**
- Amount exact + Date within ±1 day
- Reference partial match

**Tier 3 - Medium Confidence (70-89%):**
- Amount exact + Date within ±3 days
- No reference match

**Unmatched:**
- No matching system transaction found
