**User leaves with:** Selected bank account to reconcile

**Arrival:** Navigate to Cash & Bank module from main menu or dashboard notification

## Interactions

### On Load
1. Fetch all bank accounts with balances
2. Calculate reconciliation status per account
3. Check if month-end approaching
4. Display alert if reconciliation due
5. Fetch recent activity

### On "Start Reconciliation" Click
1. Navigate to Bank Account List (11.2)
2. Filter: Status = "Pending"
3. Sort by: Unreconciled Items DESC

### On "Reconcile →" Click (Account Card)
1. Navigate to Import Statement page (11.3)
2. Pre-select this bank account
3. Pass account ID as parameter

### On "Import Statement" Click
1. Navigate to Import Statement page (11.3)
2. Pre-select this bank account
3. Open file upload dialog

### On "View Transactions" Click
1. Navigate to Bank Register page
2. Filter by this account
3. Show all transactions (reconciled + unreconciled)

### On "+ Add Account" Click
1. Open "Add Bank Account" modal
2. Form: Bank Name, Account Number, Opening Balance
3. Save and refresh list

### On Period Change
1. Update all widgets for selected period
2. Refresh account balances
3. Refresh reconciliation status

### On Account Card Hover
1. Highlight card
2. Show quick actions tooltip

- Must be valid fiscal period
- Cannot select future periods
- Default: Current month

- Can be negative (overdraft)

## Edge Cases

**No Unreconciled Accounts:**
- Hide alert box
- Show success message: "✓ All accounts reconciled"
- Display last reconciliation date

**Overdue Reconciliation:**
- Red alert box
- Urgent icon
- Text: "⚠️ OVERDUE - Reconciliation past due date"

**No Bank Accounts:**
- Show empty state
- Message: "No bank accounts configured"
- Action: "+ Add Your First Bank Account"

**Negative Balance (Overdraft):**
- Display balance in red
- Show warning icon
- Note: "Overdraft"

**Large Number of Accounts:**
- Show top 3 pending accounts
- Link: "View All X Accounts →"
- Pagination in full list view

## Notes

**Design Decisions:**
- Alert box is prominent (month-end critical)
- Summary cards show at-a-glance status
- Account cards show just enough detail to decide priority
- Quick actions on each card (no need to navigate away)

**Why This Works:**
- Sari sees urgent reconciliation need immediately
- Can start reconciliation with 1 click
- Clear visibility of which accounts need attention
- Summary cards provide context without overwhelming
