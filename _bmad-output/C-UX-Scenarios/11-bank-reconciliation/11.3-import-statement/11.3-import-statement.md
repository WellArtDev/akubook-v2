# Page 11.3: Import Bank Statement

**Scenario:** 11 - Bank Reconciliation  
**Page:** 11.3 - Import Statement  
**Created:** 2026-05-13  
**Status:** ✅ Specified

---

## Page Purpose

Upload and parse bank statement CSV/Excel file

---

## Key Components

### 1. Account Selection
- Dropdown: Select bank account
- Pre-selected if coming from account card

### 2. File Upload
- Drag-and-drop zone
- File picker button
- Supported formats: CSV, Excel (.xlsx, .xls)
- Max size: 10MB

### 3. Format Detection
- Auto-detect bank format (BCA, Mandiri, BNI templates)
- Manual format selection if auto-detect fails
- Preview column mapping

### 4. Transaction Preview
- Table showing first 10 transactions
- Columns: Date | Description | Debit | Credit | Balance
- Validation indicators (green checkmarks, red errors)

### 5. Validation Results
- Total transactions found
- Date range detected
- Duplicate transactions warning
- Missing data errors

### 6. Actions
- Parse & Continue → Navigate to Auto-Match (11.4)
- Cancel → Return to dashboard
- Download Template → Get sample CSV

---

## Interactions

- File drop → Auto-detect format → Show preview
- Format change → Re-parse file → Update preview
- Validation errors → Highlight issues → Prevent continue
- Parse success → Navigate to Auto-Match (11.4)

---

## Data API

```
POST /api/cash-bank/import/parse
Content-Type: multipart/form-data

Request:
- account_id
- file (CSV/Excel)
- format (optional)

Response:
{
  "transactions": [...],
  "total": 45,
  "date_range": {"from": "2026-05-01", "to": "2026-05-31"},
  "duplicates": 2,
  "errors": []
}
```
