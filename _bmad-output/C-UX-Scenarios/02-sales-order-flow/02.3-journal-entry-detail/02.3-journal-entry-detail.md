## Page Purpose

**What it does:** Shows complete journal entry with full audit trail and review actions  
**User arrives:** From dashboard or entry list, needs to verify and approve entry  
**User leaves with:** Entry marked as reviewed/approved, confidence in accuracy

---

## Entry Context

**Device:** Desktop (office workstation)  
**User State:** Sari needs to verify entry details, check audit trail, mark as reviewed  
**Arrival:** Click "Review →" from dashboard or "View Details →" from entry list

## Interactions

### On Load
1. Fetch entry details by ID
2. Fetch transaction chain
3. Fetch audit trail
4. Display all sections
5. Focus on review notes if status = Pending Review

### On "View SO/DO/INV" Click
1. Open source document in modal
2. Show full document details
3. Option to navigate to source module

### On Account Code Click
1. Show account details in tooltip
2. Option to view account ledger

### On "Mark as Reviewed" Click
1. Validate: User has permission
2. Save review notes (if any)
3. Update status to "Reviewed"
4. Add audit trail entry
5. Show success message
6. Return to list or stay on page (user choice)

### On "Approve & Lock" Click
1. Show confirmation modal
2. Validate: User has permission
3. Save review notes (if any)
4. Update status to "Approved"
5. Lock entry (no further edits)
6. Add audit trail entry
7. Show success message
8. Return to list

### On "Reject" Click
1. Show rejection modal
2. Require rejection reason (textarea)
3. Validate: User has permission
4. Save rejection reason
5. Update status to "Rejected"
6. Add audit trail entry
7. Notify relevant users
8. Return to list

### On "Print Entry" Click
1. Generate PDF with all entry details
2. Open print dialog
3. Option to save as PDF

### On "Email to Auditor" Click
1. Open email modal
2. Pre-fill: Recipient (auditor email), Subject, Body
3. Attach entry PDF
4. Send email
5. Add audit trail entry

### On "View in GL" Click
1. Navigate to General Ledger module
2. Filter by this entry ID
3. Highlight entry in ledger

## Edge Cases

**Unbalanced Entry:**
- Show red X in total row
- Highlight difference amount in red
- Disable "Approve & Lock" button
- Show warning: "⚠️ Entry tidak balance - tidak bisa diapprove"

**Missing Transaction Chain:**
- Show: "Transaction chain not available"
- Still allow review based on entry details

**Long Audit Trail:**
- Show last 5 entries
- "Show All" button to expand

**Already Reviewed:**
- Show reviewer name and timestamp
- Show review notes
- Disable review buttons
- Show "Already reviewed by [name]"

**Locked Entry:**
- Show lock icon
- Disable all edit actions
- Show "Entry locked - approved by [name]"
