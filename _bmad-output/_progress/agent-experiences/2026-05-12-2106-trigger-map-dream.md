- All-in-one ERP for Indonesian SMEs (100+ employees, Rp 1M+ revenue/month)
- One-time payment model vs subscription fatigue
- On-premise option vs cloud lock-in
- Target: Distributors, retailers, service companies

**User Archetypes:**
1. Finance Admin / Accounting (Primary)
2. HRD Manager (Primary)
3. Purchasing Manager (Primary)
4. Sales Team (Secondary)
5. Warehouse Staff (Secondary)
6. General Staff (Secondary)

**Strategic Direction:**
- Freedom: One-time payment, on-premise, data ownership
- Simplicity: Single dashboard, eliminate tool sprawl
- Completeness: Accounting + Attendance + HRM

**Constraints:**
- Must integrate with existing hardware (ZKTeco)
- Must support data migration from Accurate/Excel
- Must work on-premise (SQLite/PostgreSQL)
